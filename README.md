Hospital Management System SQL Project

This project is a complete end-to-end Hospital Management System (HMS) built using PostgreSQL/MySQL, featuring a fully normalized relational schema, realistic synthetic data (1M+ records), and 50+ analytical SQL queries. It simulates real hospital operations including patients, doctors, appointments, admissions, treatments, lab tests, prescriptions, billing, and invoices. The dataset and queries are designed for learning database design, SQL analytics, performance tuning, and real-world data modeling.
